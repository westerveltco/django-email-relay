## Requirements

- Python 3.8, 3.9, 3.10, 3.11, 3.12
- Django 3.2, 4.2, 5.0

## Getting Started

1. Install the package from PyPI:

```bash
python -m pip install django-email-relay
```

2. Add the app to your Django project's `INSTALLED_APPS`:

```python
INSTALLED_APPS = [
    ...,
    "django_email_relay",
    ...,
]
```
